# CARSHIPMENT Website

A Carshipment website with three pages, using HTML, CSS.

## Pages
- Home
- Services 
- Contact 

## Features
- Different background color per page
- Navigation links to access all pages
- Flexbox layout for gallery and service cards
- Image showcase of vehicles and transport
- Contact form

## How to Run
1. Fork,clone or copy all files into one folder.
2. Open `index.html` in your browser.
3. Use the navigation bar to move between pages.

## Technologies
- HTML
- CSS
